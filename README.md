📊 Loan Risk & Financial Intelligence Suite
An End-to-End Power BI Portfolio Project

🎯 Executive Summary
As part of my final-year specialization in AI and Data Science, I developed this comprehensive intelligence suite to analyze a $32B+ loan portfolio. The project bridges the gap between raw on-premises data and cloud-based business intelligence, providing actionable insights into borrower risk and historical default trends.

🏗️ Technical Architecture
This project implements a professional-grade Hybrid ETL Pipeline:

Source: Microsoft SQL Server (Local Environment).

Middleware: On-premises Data Gateway using Windows Authentication for secure cloud-to-local bridging.

ETL Engine: Power BI Dataflows used for centralized, reusable M-logic and Power Query Online transformations.

Performance: Leveraged Incremental Refresh and Data Profiling to manage large-scale datasets efficiently.

📈 Key Insights & Business Impact
Volatility Analysis: Successfully identified a 1.7x growth spike in loan volume during 2018 using Year-over-Year (YoY) DAX modeling.

Risk Segmentation: Discovered that Unemployed borrowers carry a 3.39% default rate, which is approximately 1.03% higher than Full-time employees.

Portfolio Concentration: Used Decomposition Trees to reveal that High-income earners represent $21.73B of total assets.

Demographic Trends: Mapped a 11.75% default peak in 2016, providing a historical baseline for future risk modeling.
